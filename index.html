Who’s involved?

Brain Wars is a coding competition for IT experts that allows them to demonstrate how they can solve different vital tasks using algorithms and programming languages such as Java, Python and PHP. Competitors have to pass a series of online algorithmic tasks that will be validated automatically by the code review system. The system was developed by experienced developers who judges the finalists.
